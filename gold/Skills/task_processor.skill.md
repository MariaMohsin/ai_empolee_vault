# Skill: Task Processor

## Purpose
Process markdown tasks from /Needs_Action and complete them correctly.

## Inputs
- Markdown files inside /Needs_Action

## Behavior
1. Read the task file
2. Understand what is required
3. Write the response inside the same file
4. Mark task as completed
5. Move file to /Done

## Constraints
- Do not create new tasks
- Do not act outside the vault
- Do not skip steps
